# `JAVASCRIPT`

### `new`

자바스크립트의 생성자 (클래스)  ⇒ 여러 객체들을 다를 수 있다 

### `new Swiper( );`

- 자바스크립트 문법으로 Swiper 라는 함수를 실행한다
- 소괄호 사이에 추가적인 옵션들을 넣는다.

<aside>
⭐ `new Swiper('선택자' , 옵션 );`

</aside>

1. 문자 데이터 형태로 선택자를 인수로 삽입한다 
2. 옵션을 객체 데이터 형식으로 넣어준다  `{ }`  / 객체 데이터 사이에 실제 데이터 작성 
    
    `new Swiper('CSS 선택자' , { } );`    
    
    ![Untitled](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/073f49f2-e44e-4014-8a24-76b1158db94c/Untitled.png)
    
                                                                          🔻
    
    ![Untitled](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/b0afd8c6-bed8-4afe-84eb-1712b39960cc/Untitled.png)
    
    `direction`   - 방향은 수직으로  / 마우스로 클릭해서 올리면 다음 슬라이드가 나온다 
    
    `autoplay` - 자동 재새 여부 : 자동으로 슬라이드를 재생해서 클릭 해 올릴 필요가 없다 
    
    `loop` - 반복 재생 여부 : 4 번재 슬라이드 다음으로 다시 첫 번째 슬라이드가 나오게 한다
    
    ```jsx
    new Swiper('.notice-line .swiper-container' , {
      direction: 'vertical', 
      autoplay: true,        
      loop: true             
    });
    ```
